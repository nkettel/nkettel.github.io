My Development Journey Through Projects

These projects show my efforts, knowledge, and skills made visual through code. These projects made me think deeply and tested my problem solving capabilities. In each one, I had to come up with ideas, turn those ideas into python code, playtest them and make sure that they run smoothly and error free. It was quite the journey for some of them, but I really enjoyed it.

What these projects taught me:

  • Game Physics & Collision Detection: Understanding positions and real-time collision detection, demonstrated in Golf Game project
  • Code You Can Build On: Good design decisions allow for more features to be added
  • User Experience is Critical: The best code means nothing if users can't or won't use it
  • Testing is Essential: Playtesting as you develop catches bugs early, helps understand project direction, and builds confidence
  • Documentation Saves Time: Code complexity compounds quickly, but documenting helps you stay organized
  • Debugging is a Skill: Learning effective debugging techniques is as important as learning languages
  • Collaboration Improves Code: It's essential to have someone to turn to in case you get stuck
  • Performance Matters: Optimization requires a lot of brain effort, but will be needed if you want anyone to be impressed

The following projects were written in object-oriented Python code using CMU CS Academy.

Tank Battle Game

A 4-player competitive tank battle game inspired by Arras.io. Built with Python and CMU Graphics featuring real-time game physics and collision detection.

Player Controls:

Blue (P1): W/A/S/D move, E/R rotate, Q shoot
Red (P2): Y/G/H/J move, U/I rotate, T shoot
Green (P3): P/L/;/' move, [/] rotate, O shoot
Pink (P4): Arrow keys move, ./​? rotate, Enter shoot

Golf!

A 2D mini-golf inspired game demonstrating collision and movement physics. Use the three bars in the top-left to switch levels.

Controls:

Direction: Left and Right arrow
Power: Up and Down arrow
Swing:(P3): Enter

Geometry Dash

Demonstrates moving objects, collision, images, groups, and procedures to create objects in python.

Time your clicks to get through the course!

Settlers of Catan

A demake of Settlers of Catan in python demonstrating visual techniques like images, shape, and dragging objects.

Follows similar Settlers of Catan rules, but with only settlements. Drag settlements onto corners of tiles to get resources depending on dice rolls.
