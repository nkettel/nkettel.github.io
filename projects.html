My Development Journey Through Projects

These projects represent my evolution as a programmer. Starting with game development using Python and CMU Graphics, progressing through foundational Java skills for desktop applications, and expanding into web technologies like JavaScript and React. Each project challenged me in different ways and taught me valuable lessons about software design, user experience, and the importance of clean, maintainable code.

What these projects taught me:

  • Game Physics & Collision Detection: Understanding coordinate systems and real-time collision detection
  • Architecture Matters: Good design decisions early prevent technical debt later
  • User Experience is Critical: The best code means nothing if users can't or won't use it
  • Testing is Essential: Building tests as you develop catches bugs early and builds confidence
  • Documentation Saves Time: Well-documented code is significantly easier to maintain and enhance
  • Debugging is a Skill: Learning effective debugging techniques is as important as learning languages
  • Collaboration Improves Code: Code reviews and peer feedback lead to better solutions
  • Performance Matters: Writing code is just the first step; optimization and efficiency are crucial

Tank Battle Game

A 4-player competitive tank battle game inspired by Arras.io. Built with Python and CMU Graphics featuring real-time game physics and collision detection.

🎮 Game Controls:

Blue (P1): W/A/S/D move, E/R rotate, Q shoot
Red (P2): Y/G/H/J move, U/I rotate, T shoot
Green (P3): P/L/;/' move, [/] rotate, O shoot
Pink (P4): Arrow keys move, ./​/ rotate, Enter shoot
